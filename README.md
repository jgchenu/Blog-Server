# Blog-Server

jgchen 的 blog-server

### 技术栈

**前端**

- eslint
- react
- react-router
- react-redux
- redux
- axios
- antd
- es6/7/8
- jwt token 验证

**后台**

- nodejs koa sequelize mysql

此项目采用前后端分离的实现，后台接口基于 RESTful 规范设计，提供数据,基于 jwt 权限限制，前端负责路由跳转，权限限制，渲染数据等。
