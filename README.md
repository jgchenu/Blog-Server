# Blog-Server
jgchen的blog-server
### 技术栈
**前端**
* eslint
* react
* react-router
* axios
* antd
* es6/7/8

**后台**
* nodejs koa sequelize mysql

此项目采用前后端分离的实现，后台接口基于RESTful规范设计，只提供数据，前端负责路由跳转，权限限制，渲染数据等。

* [x] 博客标签

* [x] 返回顶部
